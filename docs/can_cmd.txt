Value

Meaning

Note:
●​ Every reserved field must be set to 0.
●​ For DisableBraking (bit 2): 0 = enabled, 1 = disabled.
●​ For brakingMotorLeft (bit 3): 0 = Left motor not braking, 1 = Left motor regenerative
braking active.
●​ For StandingMotorLeft (bit 4): 1 = Left motor RPM is less than MIN_RPM_STANDING (as
set in code).
●​ For StandingMotorRight (bit 5): 1 = Right motor RPM is less than MIN_RPM_STANDING
(as set in code).
●​ For brakingMotorRight (bit 6): 0 = Right motor not braking, 1 = Right motor regenerative
braking active.
